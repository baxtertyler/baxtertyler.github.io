Unarmed Knights

Project Description

Videogame created in C++ with SFML

Two clueless knights have misplaced their swords, and now an army of mischievous ghosts is out for a spooky showdown. Employ the power of multitasking as you control both knights, one with arrow keys and the other with WASD. Their only weapon? A magical rope! Can these knights survive the spirit onslaught and become the ultimate ghostbusters in this endless adventure?

Role Description

  • Sole creater of the project, generating all coding and implementation in an object-oriented format from start to finish.
  • Created a sophisticated AABB collision system to ensure accurate interactions between weapons, enemies, and obstacles, enhancing the overall gameplay experience.
  • Implemented dynamic sprite animations using sprite sheets, allowing characters and objects to respond seamlessly to the game's current state and enhancing visual appeal.
  • Untilized the SFML (Simple and Fast Multimedia Library) import to establish the start/game/end windows and capture key inputs, providing players with an intuitive and responsive gaming interface.

The Endless World Sim

Project Description

Program created in Java with Processing

This simulated program displays the creation of an infinitely expansive virtual world inhabited by interactive entities. These entities exhibited dynamic behavioral changes in response to simulated events, offering a captivating simulation experience. An ingenious A-star pathing strategy was devised and seamlessly integrated, revolutionizing entity movement and enhancing their ability to navigate the virtual landscape effectively. The project showcased a meticulous emphasis on inheritance and abstract classes, facilitating modularity and extensibility throughout the simulation's codebase.

Role Description

  • Worked to modify procedurally written single file .java program into an extensive object-oriented project based on customly desiged .yml diagram
  • Implemented an A * pathing strategy for entitites to effectively move across the 2D playgrid for smooth and efficient gameplay.
  • Created art and action for new entites based on both in-game and player-created events.
  • Synced entites with gameplay clock to ensure neverending simulation as well as new situations each run through.

Shoot the Target!!!

Project Description

Videogame created in JavaScript with Processing5

In this thrilling game, it's all about shooting those tricky targets that pop up on your screen. But beware, they like to shrink or dance around, and a wrong click means you lose points! Go for the bullseye to earn extra point and become the champion. With various difficulties and time limits, it's a race against time to score big. For the daring, there's Survival Mode: just one second to click, three lives, and your goal is simple: grab as many points as you can before you die. Get ready to click your way to glory in Shoot the Target!!

  • Developed multiple generations dependent on crowd-testing data to ensure a seamless and effective game
  • Programmed and implemented critical components including the start and pause menu, target movement, timer and lives counter, in-game currency
  • Designed art and programmed behavior for entities and animations
