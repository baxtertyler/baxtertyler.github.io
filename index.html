My Work

Crib

Crib is an innovative software engineering project designed to simplify and streamline household chore management among roommates. As the sole backend engineer on the project, I established a Rest API from the ground up using Node.js and Express.js A. Crib utilizes a cloud-based database to ensure seamless storage and retrieval of critical project data, and uses user authentication mechanisms, fortifying the platform's security and privacy features.

The Endless World Simulation

Developed in Java, the Infinite World Simulation depicts an endless simulation of numerous interactive entities that exhibit behavioral changes based on both entity and player-controlled events, enhanced with an integrated A-star pathing strategy. This program follows an object-oriented style of programming with an emphasis on inheritance and abstract classes.

SPIM Emulator

A comprehensive SPIM emulator crafted in Java for executing MIPS assembly code. Users can dynamically navigate the program, choosing to step through lines, run specific segments, or execute the entire code. Inspection of memory and register values is available at any point, and users can terminate the program or reset the register state. Advanced enhancements introduced a pipeline and branch predictor with precise addition of stalls and squashes as needed.
